# GenAI-Powered Legal Assistant for SMEs

## Problem Statement
Small and medium enterprises struggle to understand complex legal contracts, which leads to hidden risks and unfavorable obligations.

## Solution
This project is a GenAI-ready legal assistant that analyzes business contracts, identifies risky clauses, and explains them in simple business language.

## Tech Stack
- Python
- Streamlit
- Rule-based NLP
- GenAI-ready architecture (no external legal data)

## Status
Phase 1 completed: Project setup and base UI implemented.
